Kontaktformulare in HTML

Formulare helfen uns dabei, Daten von Nutzern einzusammeln. Die wichtigsten Bausteine sind:

  • <form> = der Formular Rahmen (Container)
  • <label> = Beschriftung für die Eingabefelder
  • <input> = Eingabefeld (Bsp.: Name, Passwort, E-Mail, Telefonnummer, etc.)
  • <textarea> = großes Textfeld (Bsp.: Nachrichten)
  • <select> und <option> = Dropdown Auswahl
  • <button> = Button (Absenden/ Zurücksetzen)

1. Das einfache Kontaktformular

Ein Kontaktformular besteht meistens aus Name, E-Mail und Nachricht.




for (bei <label>) + id
Verebindet Label und Feld. Klickst du auf das Label, sprint dein Cursor ins Feld

2. Wichtigen Form-Attribute: action und method

action=""
Wohin die Daten geschickt werden (Bsp.: eine Server-Adresse oder ein Skript).

method="GET"
Daten werden in die URL (= Uniform Resource Locator) geschrieben (gut für eine Suche, schlecht für unsere sensiblen Daten).

method="POST"
Daten werden im Hintergrund gesendet (Standard Formular).

HINWEIS: Ohne echten server passiert beim Absenden oft nichts Sichbares, aber HTML "baut" das Formular korrekt.

3. Input-Typen und ihre Funktionen

Der type sagt dem Browser, welche Art von Daten erwartet werden. Dadurch bekommt man die passenden Tastaturen am Handy/Tablet und manchmal auch eine automatische Überprüfung.

placeholder = ""
Ein Hinweistext im Feld (verscwindet beim Tippen).
min/max
Grenze der Zahlen (oder ein Datum)
min = minimum (kleinster wert) | max = maximum (größter wert)

4. Pflichtfelder und einfache Validierung

Mit required kann man Felder verpflichted setzen. Der Browser verhindert dann das Absenden, wenn etwas fehlt.

required
Muss ausgefüllt werden!!!!

minlength
Mindestlänge für unseren Text.

maxlength
Maximale länge für unseren Text.

5. Auswahlfelder: select, radio, checkbox

radio
Nur eine Option ist auswählbar.

checkbox
Ist variable zwischen an und aus oder mehreren Auswahlmöglichkeiten.

Kontaktwunsch:


>

disabled
Option kann nicht ausgewählt werden.

selected
Option ist standardgemäß ausgewählt.

6. Buttons: submit, reset, button

type="submit"
Sendet das Formular ab.

type="reset"
Setzt alle Felder auf den Startzustand zurück.

type="button"
Macht erst etwas, wenn JavaScript hinzugefügt wurde.
