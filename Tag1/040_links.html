Anchor-Tags (<a>) - Links in HTML

Mit den Anchor-Tags können wir Links erstellen. Links führen zu anderen Seiten, Dateien oder zu einer Stelle auf derselben Seite.

1. Grundaufbau eines Links

So sieht ein einfacher Link im Code aus:

<a href="https://www.google.com/">Ich bin ein Link</a>

So sieht ein Link im Browser aus:

Das ist ein Link

2. Das Attribut href

href steht für Hypertext Refrence. Das Attribut href gibt an, wohin der Link führt. Ohne href ist es kein richtiger Link.

  • href="./010_hallo.html" → ink zu einer Datei im gleichen Ordner
  • href="#ziel" → Link zu einer Stelle auf derselben Seite
  • href="https://www.google.com/" → Link der zu einer anderen Webseite führt

3. Link in einem neuen Tab öffnen

Mit target="_blank" öffnet sich der Link in einem neuen Tab:

<a href="https://www.google.com/" target="_blank" rel="noopener noreferrer">Ich bin ein Link</a>

So sieht ein Link im Browser aus:

Das ist ein Link

noopener Sicherheit bei target="_blank", das die Zielseite, meine Seite nicht manipulieren kann.

noreferrer Datenschutz, bedeutet die Zielseite sieht nicht, von welcher Seite der Klick kam.

rel eng. relationship / dt. Beziehung

4. E-Mail und Telefon Links

E-Mail (mailto)

<a href="mailto:beispiel@gmail.com">E-Mail schreiben</a>

Unser Browser öffnet das Standard E-Mail Programm des Nutzers. Er erstellt uns automatisch einen Empfänger (E-Mail ist schon ausgefüllt), Betreff leer, Text leer.

E-Mail schreiben

Telefon (tel)

<a href="tel:+49123456789">Jetzt anrufen</a>

Smartphones und Tablets öffnen direkt die Telefon App zum Anrufen. Pcs können die funktion nur nutzen, wenn die dafür zuständige App vorhanden ist.

Anrufen

5. Internen Links (Sprungmarken)

Du kannst innerhalb derselben Seite springen. Dafür braucht man das Ziel-Element id. 

<a href="#ziel">Springe zum Ziel</a>
Springe zum Ziel

Bei Seiten auf bestimmten Sprungmarken landen

Du kannst direkt auf einer bestimmten Seite an einer bestimmten Stelle landen, wenn diese eine entsprechende id hat. 

<a href="./030_struktur_elemente.html#hinweis">Lande auf der Seite mit den Hinweisen</a>
Lande auf der Seite mit den Hinweisen
