Listen in HTML

In diesem Abschnitt kann ich alles über verschiedene Listen nachlesen. HTML kennt:
<ol> (sortiertierte Liste) <ul> (ungeordnete Liste) und <dl> (beschreibende Liste).

1. Wann nutzen wir welche Liste?

  • <ol>: Wenn die Reinfolge wichtig (bsp.: Rangliste, Anleitung, Rezept).
  • <ul>: Wenn die Reinfolge egal ist (Bsp.: Stichpunkte, Einkaufsliste, To-Do Listen)
  • <dl>: für Begriffe mit Erklärung (Bsp.: Glossar, FAQ, Wörterbuch)

2. Sortierte Liste - <ol>

Eine sortierte Liste nummeriert die einträge automatisch

Beispiel (früh aufstehen):

  1. Augen öffnen
  2. Aufrichten
  3. Uns aus dem Bett bewegen

Wichtige Attribute

  • type: 1, A, a, I, i
  • start: Startwert festlegen
  • reversed: Nummerierung umkehren

Beispiel (Tiere):

  1. Hund
  2. Katze
  3. Kuh
  4. Elephant
  5. Meerschweinchen
  1. Hund
  2. Katze
  3. Kuh
  4. Elephant
  5. Meerschweinchen
  1. Hund
  2. Katze
  3. Kuh
  4. Elephant
  5. Meerschweinchen

3. Unsortierte Liste (<ul>)

Eine unsortierte Liste stellt Aufzählungen dar, bei der die Reihenfolge egal ist.

Beispiel (Städte):

  • Madrid
  • London
  • Paris

Wichtige Attribute

  • type: square, circle, none

Beispiel (Städte):

  • Madrid
  • London
  • Paris
  • Madrid
  • London
  • Paris
  • Madrid
  • London
  • Paris

4. Beschreibende Listen (<dl> = Description List)

Eine beschreibende Liste besteht aus: <dt> (Term = Begriff) und <dd> (Description = Beschreibung).

Beispiel (Begriffe in der IT):

HTML
Auszeichnungssprache zur Strukturierung der Webseite.

CSS
Sprache für das Design und Layout der Webseite.

Mehrere Beschreibungen für einen Begriff

Dackel
Hunderasse
Gilt als intelligent, lernfähig und stur.

5. Verschachtelte Listen

Listen können in anderen Listen eingebettet werden (Bsp.: Menüs oder Kategorien).

Beispiel (Elektro Geräte):

  • Produkt
    • Handys
      • Androids
      • iOS

Anleitung: Verschachtelte Listen in HTML

  1. Erstelle zuerst eine normale Liste mit <ul> oder <ol>
  2. Lege einen Listeneintrag mit <li> an
  3. Schreibe den Text für den Haupteintrag in dieses <li>
  4. Öffne innerhalb desselben <li> eine neue Liste (<ul> oder <ol>)
  5. Lege in dieser neuen Liste die Unterpunkte mit <li> an
  6. Schließe zuerst die Unterliste und danach das äußere <li>
